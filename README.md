# 图书管理系统 python810

## 项目概述

图书管理系统是基于`PyCharm`开发环境，采用`Flask`框架与`Python`语言构建的一个高效、易用的管理系统。

## 技术栈

- **后端**：Python 3.x，Flask
- **数据库**：SQLite
- **开发工具**：PyCharm

## 功能模块

系统主要功能模块包括：

- 图书信息管理
- 读者信息管理
- 借阅与归还处理
- 系统用户权限管理

（具体功能以实际图片详情为准）

## 系统角色

- 系统管理员
- 普通用户（读者）

## 运行环境

- Python 3.x
- Flask
- 支持现代浏览器

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 配置数据库SQLite
4. 部署应用到服务器

## 目录结构

```
project/
│
├── app/                # Flask应用目录
│   ├── models/         # 数据模型目录
│   ├── views/          # 视图层目录
│   ├── templates/      # HTML模板目录
│   └── static/         # 静态资源目录
│
├── venv/               # Python虚拟环境
├── requirements.txt    # 依赖列表
└── run.py              # 应用启动脚本
```

## 核心亮点

- **简洁的代码**：基于Python语言，易于理解和维护。
- **灵活的框架**：使用Flask框架，方便进行二次开发与功能扩展。
